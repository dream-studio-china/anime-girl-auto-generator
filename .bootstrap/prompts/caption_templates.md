
```

```

## Hashtag 策略

原则: hashtag 必须与图片内容相关，单帖建议 3-6 个。不要为了蹭流量加入无关热门标签，不要多帖重复相同标签组合。

### Tier 1 — 必加 (高流量基础标签)
```
#animegirl #AIart #waifu #anime #animeillustration
```

### Tier 2 — 时段热门 (每天根据趋势调整)
```
#animeart #digitalart #animeartwork #characterdesign
```

### Tier 3 — 风格定向 (按图内容选择)
| 风格 | 标签 |
|------|------|
| 赛博朋克 | `#cyberpunk #cyberpunkart` |
| 和风 | `#japaneseart #kimono #和風` |
| 性感/擦边 | `#sexyanime #waifumaterial`，发布前必须确认成人/敏感媒体设置 |
| 可爱 | `#kawaii #moe #cuteanime` |
| 暗黑/哥特 | `#gothicanime #darkfantasy` |
| 日常 | `#everydayanime #sliceoflife` |
| 泳装/夏日 | `#swimsuit #summeranime` |
| 制服 | `#schoolgirl #uniform` |

### Tier 4 — AI 社区

## Caption 生成规则

从 prompt 提取角色关键词 + 氛围词 → 组合 caption。标注 ComfyUI 来源。参考上方模板即可，不要每次都读取完整配额/时区表。

## 示例

**输入:** workflow=yume_no_girl_x, prompt含 "blonde ponytail girl, sunset beach, bikini"
**自动生成 caption:**
```
夏の夕暮れ、金色のポニーテール 🌅
Golden hour by the sea — summer vibes
#animegirl #swimsuit #summeranime #AIart #ComfyUI
```
