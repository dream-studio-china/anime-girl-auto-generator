## Caption 生成规则

Agent 在发布时自动生成 caption，按以下优先级组合:

1. **从 prompt 提取角色关键词** → character_desc (如 "silver hair girl in cyberpunk city")
2. **判断画面氛围** → mood_keyword (如 "mysterious", "cute", "elegant")
3. **根据当前 X 趋势** → 加入 trending hashtag
4. **根据受众时段** → 选择 language (单语/双语)
5. **标注 ComfyUI 来源** → 增加技术标签
